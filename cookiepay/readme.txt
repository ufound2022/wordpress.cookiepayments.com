== Description ==

쿠키페이는 복잡한 코드 없이 손쉽게 PG결제 연동이 가능한 플랫폼입니다.
손쉬운 개발소스, 직관적 UI, 샘플 예제를 제공하여 가맹점이 직접 결제 연동이 가능하도록 지원하고 있습니다.
1.5%의 국내 최저 결제 수수료는 물론 3일 주기의 빠른 정산진행으로 가맹점의 사업 성공을 위해 노력 합니다.

쿠키페이는 비대면 결제전문가와 IT시스템이 결합된 통합결제솔루션으로 PG이용의
한계를 탈피하여 고객 중심의 서비스를 제공합니다.

서비스 소개: https://cookiepayments.com
데모 페이지 : https://wordpress.iroholdings.co.kr/


== Installation ==

쿠키페이 플러그인 설치, 사용을 위해서는 https://cookiepayments.com/page/form에서 회원가입해야합니다.

1. 플러그인 파일cookiepay.zip파일을 다운받은 후 압축을 풀어 `/wp-content/plugins/` 디렉토리에 복사합니다.
2. 압축 파일을 해제하면 cookiepay폴더가 생성됩니다.
3. 워드프레스 관리자페이지에서 'Plugins'메뉴를 통해 "cookiepay for woocommerce" 플러그인을 활성화합니다.
4. https://cookiepayments.com/page/form에서 회원가입 후 설정합니다.
5. 우커머스(woocommerce) > 설정 > 결제 페이지에서 쿠키페이 결제를 활성화한 후 '관리'페이지에서 설정을 마무리합니다.
